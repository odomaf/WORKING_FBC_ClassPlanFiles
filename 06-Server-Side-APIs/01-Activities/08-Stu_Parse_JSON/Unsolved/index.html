Parsing JSON (Unsolved)

Your task is to use fetch() to request Star Wars characters from https://swapi.dev/api/people/, parse the JSON, and log the first 5 characters’ name and birth_year to the console.

  1. Create requestUrl in assets/js/script.js.
  2. Call fetch(requestUrl) → .then(response.json()) → .then(data).
  3. Inspect the raw data in DevTools. Use data.results.
  4. Loop through the first five items (indexes 0–4).
  5. Log name and birth_year for each.

Open your browser console to see your output.

    Use let/const, no arrow functions, and add a .catch() if you’d like.
