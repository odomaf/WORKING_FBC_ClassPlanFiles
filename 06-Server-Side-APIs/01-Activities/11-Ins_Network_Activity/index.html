Server-Side APIs

Network Activity Tab

In this demo, we send one successful request and one failing request to the Star Wars API (SWAPI).

Use this to practice reading the Network tab in DevTools:

  1. Open DevTools (Cmd+Opt+I on Mac, Ctrl+Shift+I on Windows).
  2. Click the Network tab.
  3. Reload the page (Cmd/Ctrl + R).
  4. Filter by Fetch/XHR to see the API calls.
  5. Compare the Status codes (e.g., 200 vs 404).

Status codes quickly tell you what happened with each request.
