# 02-Stu_Curl_Debug

## Goal

Fix a broken `curl` command to a public API.

## Task

The command below prints nothing. Why?

curl http://api.chucknorris.io/jokes/random

## Expected

You should see JSON with a "value" field.

## Hint

- Check the protocol: `http` vs `https`.
- Compare with the API docs home page: https://api.chucknorris.io/
